# Sarkara Layout

A React layout package based on Chakra UI.

## Install

```sh
npm install @cieloazul310/sarkara

// or indivisual
npm install @cieloazul310/sarkara-layout
```

### Peer dependencies (required)

- [react]
- [react-dom]
- [@chakra-ui/react]
- [@chakra-ui/icons]
- [@cieloazul310/sarkara-components]
- [@emotion/react]
- [@emotion/styled]
- [framer-motion]

## API

### Layouts

#### `<SarkaraLayout>`

Basic layout structure.

#### `<SarkaraLayoutLite>`

Sarkara layout with placeholders for header, jumbotron and footer.

### Layout components

#### `<SarkaraHeader>`

#### `<SarkaraFooter>`

#### `<DrawerContents>`
