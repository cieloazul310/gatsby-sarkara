# Sarkara

Sarkara theme package for React based on Chakra UI.

## Install

```sh
npm install @cieloazul310/sarkara
```

### Peer dependencies (required)

- [react]
- [react-dom]
- [@chakra-ui/react]
- [@chakra-ui/icons]
- [@emotion/react]
- [@emotion/styled]
- [framer-motion]
