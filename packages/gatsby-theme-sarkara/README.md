# Gatsby Theme Sarkara

Gatsby theme package based on Chakra UI.

## Install

```sh
npm install @cieloazul310/gatsby-theme-sarkara
```

### Peer dependencies (required)

- [react]
- [react-dom]
- [gatsby]
- [@chakra-ui/react]
- [@chakra-ui/icons]
- [@emotion/react]
- [@emotion/styled]
- [framer-motion]
